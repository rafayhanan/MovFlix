# MovFlix
A movie listing mobile app made using flutter and TMDB API.

## Prerequisites
- Flutter SDK
- Firebase 

## Getting Started

1. Clone the Repository

   ```sh
    git clone https://github.com/rafayhanan/MovFlix.git
    ```

2. Install Dependencies

   ```sh
    flutter pub get
    flutter doctor
    ```
3. Run

   ```sh
    flutter run main.dart
    ```
